Unit 15

Learning Aim: A

Understand the benefits and implications of installing and maintaining software in technology systems.

I had to learn about the different types of software such as system, application and programming software. I also had to learn about the purpose of installing and maintaining software, as well as benefits and implications.

Learning Aim: B

Plan installation and maintenance of software in a technology system.

I had to create a plan for the isntallation and maintenance of software.

Learning Aim: C

Install, maintain and test software in a technology system.

Following the plan I created, I need to perform the installation and maintainace activities to the user requirements.

Learning Aim: D

Review the modified technology system.

After the installation and maintaining process, I have to review the modified system, justifying changes made and mentioning any constraints.
