Unit 13 - Website Development

Learning Aim: A

Understand the uses and features of websites.

I investigated 2 websites to review the intended purpose of their differnet features and how they improve user experience

Learning Aim: B

Design a website.

I was tasked to design a website for Mobile World Station to promote mobile phones, accessories and services, with the main requirements of it being easy to use and relevant to the target audience.

Learning Aim: C

Develop and test a website.

I had to delevop my website following the design, then testing the completed website. I had to document any changes made from the original design.

Learning Aim: D

Review the finished website.

I had to gather feedback from others about my finished website in order to review my website.
